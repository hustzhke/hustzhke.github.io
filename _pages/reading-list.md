> 以下书单是为对金融、投资、机器学习及商业管理感兴趣的学生和研究者整理的入门指南，按主题分类。

---

## 金融与投资

| 主题 | 推荐书目 | 核心内容 |
|------|----------|----------|
| 如何入门投资学 | 《投资学》（滋维·博迪 等） | 建立收益、风险、资产配置和投资组合的基础框架 |
| 如何理解价值投资 | 《聪明的投资者》（本杰明·格雷厄姆） | 建立安全边际、市场先生和长期投资思维 |
| 如何理解衍生品 | 《期权、期货及其他衍生产品》（约翰·赫尔） | 学习期权、期货、互换和风险管理 |

## 量化投资与金融工程

| 主题 | 推荐书目 | 核心内容 |
|------|----------|----------|
| 如何理解金融数据 | 《金融时间序列分析》（Ruey S. Tsay） | 学习收益率、波动率、相关性和金融建模方法 |
| 如何学习因子投资 | 《主动投资组合管理》（Grinold & Kahn） | 理解 Alpha、风险模型、组合构建和主动管理 |
| 如何理解金融机器学习 | *Advances in Financial Machine Learning*（Marcos López de Prado） | 学习金融场景下的机器学习方法 |
| 如何入门算法交易 | *Algorithmic Trading*（Ernie Chan） | 理解交易策略设计、回测和执行 |
| 如何理解量化基金 | *Inside the Black Box*（Rishi K. Narang） | 了解量化基金的研究、交易和风控流程 |
| 量化面试：数学与概率基础 | 🟢 *A Practical Guide to Quantitative Finance Interviews*（绿皮书，Xinfeng Zhou） | 概率、脑筋急转弯、统计、线代、期权基础 |
| 量化面试：华尔街真题训练 | 🔴 *Heard on the Street*（红皮书，Timothy Crack） | 华尔街量化面试经典题库 |
| 量化面试：Quant 概念与问答 | 🟡 *Frequently Asked Questions in Quantitative Finance*（黄皮书，Paul Wilmott） | Quant finance 常见概念问答 |

## AI 与数据科学

| 主题 | 推荐书目 | 核心内容 |
|------|----------|----------|
| 如何学习 Python 数据分析 | [*Python for Data Analysis*](https://wesmckinney.com/book/)（Wes McKinney） | 掌握 Pandas、NumPy 和数据清洗方法，作者官方免费在线版 |

| 如何理解机器学习系统 | *Designing Machine Learning Systems*（Chip Huyen） | 理解数据、训练、部署和监控 |

## 阅读建议

1. **从基础到进阶**：建议先读《投资学》《管理学》建立金融与管理的基础框架，再按兴趣深入量化、AI 或创业方向。
2. **理论结合实践**：每读完一本书，尝试用 Python 复现案例（如《金融时间序列分析》+ 真实数据；*Build a Large Language Model* + PyTorch 实现）。
3. **跨学科连接**：金融 ML 不是简单算法套用，需理解金融数据的独特性质（低信噪比、非平稳）；管理决策需结合行为心理学（《影响力》《思考，快与慢》）。
4. **面试准备**：量化方向建议绿皮书 → 红皮书 → 黄皮书顺序刷题，配合《主动投资组合管理》的理论框架。

> 对以上方向感兴趣的学生，欢迎与我联系讨论阅读计划和科研方向。
